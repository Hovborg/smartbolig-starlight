Never forget the laundry again! This guide shows how to get a notification on your phone (or via speaker) when the washing machine is done - based on power consumption.

![Washing machine automation overview](/images/guides/vaskemaskine-notification/overview.svg)

## ü§î How Does It Work?

Washing machines use a lot of power during washing (500-2000W) and almost none when finished (0-5W). By measuring power consumption we can detect:

| State | Power Usage | Description |
|-------|-------------|-------------|
| **Off** | 0W | Machine is off |
| **Standby** | 1-5W | On but not active |
| **Running** | 100-2000W | Active wash/spin |
| **Done** | 1-5W (after high) | Was high, now low = done! |

<Aside type="tip" title="Simple principle">
When power consumption drops from high (over 10W) to low (under 5W) for more than 3 minutes = washing machine is done!
</Aside>

## üîß Setup

### Step 1: Connect Smart Plug

<Steps>
1. Plug smart plug into outlet
2. Connect washing machine to smart plug
3. Add plug to Home Assistant
4. Verify you have a **power sensor** (W)
</Steps>

### Step 2: Find Your Values

Run a wash and observe power consumption in Home Assistant.

**Typical values:**
```
Standby:        ~2W
Filling:        ~50W
Washing:        ~200-500W
Spinning:       ~300-600W
Done:           ~2W
```

### Step 3: Create Helper (Optional but Recommended)

<Steps>
1. Go to **Settings** ‚Üí **Devices & services** ‚Üí **Helpers**
2. Click **Create helper** ‚Üí **Dropdown**
3. **Name**: "Washing Machine Status"
4. **Options**: `off`, `running`, `done`, `emptied`
</Steps>

## ü§ñ Automation: Basic Notification

### Simple Version (Recommended Start)

```yaml
automation:
  - alias: "Washing machine done notification"
    description: "Send message when washing machine is done"
    trigger:
      - platform: numeric_state
        entity_id: sensor.washing_machine_power
        below: 5
        for:
          minutes: 3
    condition:
      - condition: state
        entity_id: input_select.washing_machine_status
        state: "running"
    action:
      - service: notify.mobile_app_your_phone
        data:
          title: "üß∫ Washing Machine"
          message: "The laundry is done!"
          data:
            tag: "washing_machine"
```

### Better Version with Status Tracking

```yaml
automation:
  # When machine starts
  - alias: "Washing Machine: Started"
    trigger:
      - platform: numeric_state
        entity_id: sensor.washing_machine_power
        above: 10
        for:
          minutes: 1
    action:
      - service: input_select.select_option
        target:
          entity_id: input_select.washing_machine_status
        data:
          option: "running"

  # When machine is done
  - alias: "Washing Machine: Done"
    trigger:
      - platform: numeric_state
        entity_id: sensor.washing_machine_power
        below: 5
        for:
          minutes: 3
    condition:
      - condition: state
        entity_id: input_select.washing_machine_status
        state: "running"
    action:
      - service: input_select.select_option
        target:
          entity_id: input_select.washing_machine_status
        data:
          option: "done"
      - service: notify.mobile_app_your_phone
        data:
          title: "üß∫ Washing Machine"
          message: "The laundry is done! Remember to empty the machine."
          data:
            tag: "washing_machine"
            persistent: true
            actions:
              - action: "WASHING_EMPTIED"
                title: "‚úì Emptied"
```

## üì¢ Advanced: Repeated Reminders

If you (like most people) ignore the first notification:

```yaml
automation:
  - alias: "Washing Machine: Repeat reminder"
    trigger:
      - platform: time_pattern
        minutes: "/15"  # Every 15 minutes
    condition:
      - condition: state
        entity_id: input_select.washing_machine_status
        state: "done"
    action:
      - service: notify.mobile_app_your_phone
        data:
          title: "üß∫ Washing Machine - REMINDER"
          message: "The laundry is still waiting!"
          data:
            tag: "washing_machine"
```

## üîß Troubleshooting

<Aside type="tip" title="Notification comes too early?">
Washing machines can pause during wash. Increase `for` time to 5-10 minutes.
</Aside>

<Aside type="tip" title="Notification doesn't come?">
1. Check that power sensor updates (may be delayed)
2. Verify trigger value (5W) matches your machine
3. Check condition - maybe status isn't "running"
</Aside>
