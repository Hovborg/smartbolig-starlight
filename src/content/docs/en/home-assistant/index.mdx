### Which method should I choose?

| Method | Best for | Difficulty | Add-ons |
|--------|----------|------------|---------|
| **Raspberry Pi** | Beginners, dedicated device | ⭐ Easy | ✅ Yes |
| **Proxmox VM** | Home server, homelab | ⭐⭐ Medium | ✅ Yes |
| **Docker** | NAS, existing server | ⭐⭐⭐ Advanced | ❌ No |

<Aside type="tip" title="Not sure?">
- **Completely new?** → Raspberry Pi
- **Have a Proxmox server?** → Proxmox VM
- **Have NAS/Docker?** → Docker
</Aside>
