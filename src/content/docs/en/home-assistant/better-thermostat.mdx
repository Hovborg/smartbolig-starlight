Better Thermostat is a HACS integration that makes your "dumb" radiator thermostats smart. It uses an external temperature sensor, window sensors and weather data to optimize heating and save energy.

![Better Thermostat overview](/images/guides/better-thermostat/overview.svg)

<Aside type="note" title="Open Source">
Better Thermostat is open source! Check out the [Better Thermostat GitHub](https://github.com/KartoffelToby/better_thermostat) for documentation and feature requests.
</Aside>

Radiator thermostats (TRV) measure temperature directly at the radiator - which is typically 2-5Â°C warmer than the rest of the room. Better Thermostat solves this:

| Problem | Better Thermostat Solution |
|---------|---------------------------|

| **Open window = waste** | Automatic stop on open window |
| **Manual adjustment** | Presets: Eco, Comfort, Away, Boost |
| **Multiple radiators** | Group TRVs in same room |
| **Calibration** | Automatic offset calibration |

<Aside type="tip" title="Energy savings">
By using an external sensor and window detection, you can typically save 15-25% on heating bills!
</Aside>

### Switch Preset Based on Presence

```yaml
automation:
  - alias: "Heating: Away when no one home"
    trigger:
      - platform: state
        entity_id: group.people_home
        to: "not_home"
        for:
          minutes: 30
    action:
      - service: climate.set_preset_mode
        target:
          entity_id: 
            - climate.living_room_heating
            - climate.bedroom_heating
        data:
          preset_mode: away

  - alias: "Heating: Comfort when home"
    trigger:
      - platform: state
        entity_id: group.people_home
        to: "home"
    action:
      - service: climate.set_preset_mode
        target:
          entity_id: 
            - climate.living_room_heating
            - climate.bedroom_heating
        data:
          preset_mode: comfort
```

## ğŸªŸ Window Detection

Better Thermostat can automatically stop heating when a window opens:

### With Physical Sensor

```yaml
window_sensor: binary_sensor.window_living_room
window_delay: 30  # Wait 30 sec before reacting
```

### Without Sensor (Temperature-based)

Better Thermostat can detect open windows via sudden temperature drop:

```yaml
window_detection_method: temperature_drop
window_detection_threshold: 0.5  # Â°C drop over short time
```

## ğŸ”§ Calibration

Better Thermostat automatically calibrates your TRV based on the difference between the TRV's built-in sensor and your external sensor.

### How It Works

1. TRV measures 24Â°C (at radiator)
2. External sensor measures 21Â°C (in room)
3. Better Thermostat calculates offset: -3Â°C
4. Sends calibration to TRV

## ğŸ”§ Troubleshooting

<Aside type="tip" title="TRV not responding?">
1. Check that TRV is online in Home Assistant
2. Verify that Better Thermostat uses correct entity
3. Check that TRV supports temperature calibration
4. Restart Better Thermostat integration
</Aside>

<Aside type="tip" title="Temperature jumping?">
- Reduce update frequency on external sensor
- Enable "smooth temperature" in Options
- Check sensor isn't affected by sun/draft
</Aside>
