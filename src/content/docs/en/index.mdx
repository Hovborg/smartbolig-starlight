## 🧭 Quick Start Path

<Aside type="note" title="Recommended order">
  1. **Choose platform**: Raspberry Pi (easy), Proxmox (advanced), or Docker (existing server).
  2. **Add Zigbee**: Zigbee2MQTT for sensors and switches from IKEA, Aqara etc.
  3. **Build your own devices**: ESP32 with ESPHome for temperature, motion, LED strips.
  4. **Automate**: Start with simple automations and expand gradually.
</Aside>
