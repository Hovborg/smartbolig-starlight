This part of SmartBolig.net covers AI tools as a separate topic. It is not mixed into Home Assistant, ESP32, products, or smart home guides.

The first focus is **AI CLIs**: terminal tools that can read code, explain projects, write changes, and run commands with your permissions.

<Aside type="caution" title="Start safely">
AI CLIs can access files, terminal commands, and sometimes the network. Start in a Git repository, check `git status`, and begin with read-only or explanation tasks before allowing edits.
</Aside>

## Recommended order

1. Read [AI CLIs: getting started](/en/ai/ai-cli/)
2. Pick one CLI and run a read-only first task.
3. Read [security and permissions](/en/ai/ai-cli/sikkerhed-og-permissions/) before allowing edits.
4. Read [pricing and plans](/en/ai/ai-cli/priser-og-planer/) if you plan to use API keys or work heavily in large repositories.
5. Add a project instruction file when you have a workflow that works.
6. Follow [AI News](/en/ai/nyheder/) for model, CLI, API pricing, and security changes.
