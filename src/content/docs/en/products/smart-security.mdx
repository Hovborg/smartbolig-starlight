## üö® Alarm Systems

### DIY vs Professional

| Type | Pros | Cons | Examples |
|------|------|------|----------|
| **DIY (Home Assistant)** | Cheap, flexible, local | Requires setup, no monitoring | Zigbee sensors + HA |
| **Hybrid** | Pro hardware + local | More expensive | Yale Sync, Ring Alarm |
| **Professional** | 24/7 monitoring | Expensive subscription | ADT, SimpliSafe |

### DIY Alarm with Home Assistant

Build your own alarm with Zigbee sensors:

**Components:**
- Door/window sensors (Aqara, Sonoff)
- Motion sensors (PIR or mmWave)
- Siren (Zigbee or WiFi)
- Keypad (Zigbee or smart lock keypad)

```yaml
# Home Assistant Alarm Panel
alarm_control_panel:
  - platform: manual
    name: Home Alarm
    arming_time: 30
    delay_time: 30
    trigger_time: 120
    disarmed:
      trigger_time: 0
    armed_home:
      arming_time: 0
      delay_time: 0

# Automation: Trigger alarm
automation:
  - alias: "Alarm trigger on intrusion"
    trigger:
      - platform: state
        entity_id: 
          - binary_sensor.front_door
          - binary_sensor.living_room_window
          - binary_sensor.back_door
        to: "on"
    condition:
      - condition: state
        entity_id: alarm_control_panel.home_alarm
        state: "armed_away"
    action:
      - service: alarm_control_panel.alarm_trigger
        entity_id: alarm_control_panel.home_alarm
      - service: notify.mobile_app
        data:
          title: "‚ö†Ô∏è ALARM!"
          message: "Intrusion detected!"
```

## üé¨ Frigate NVR

For advanced AI-based video analysis, use **Frigate**:

### What is Frigate?

- ‚úÖ Open source NVR for Home Assistant
- ‚úÖ Real-time object detection (person, car, pet)
- ‚úÖ Google Coral TPU support
- ‚úÖ Runs locally (no cloud)
- ‚úÖ Integrates with all RTSP cameras

### Hardware Requirements

| Component | Recommended |
|-----------|-------------|
| **Server** | Dedicated mini-PC or NAS |
| **AI Accelerator** | Google Coral USB (~$70) |
| **Storage** | SSD for database, HDD for recordings |
| **RAM** | Min. 4GB |

### Basic Frigate Config

```yaml
# frigate.yml
mqtt:
  host: 192.168.1.100

cameras:
  driveway:
    ffmpeg:
      inputs:
        - path: rtsp://admin:password@192.168.1.50:554/h264Preview_01_main
          roles:
            - detect
            - record
    detect:
      width: 1920
      height: 1080
      fps: 5

detectors:
  coral:
    type: edgetpu
    device: usb

objects:
  track:
    - person
    - car
    - dog
    - cat
```
