<Aside type="tip" title="Top Recommendations 2025">

- **Best value**: Fire TV Stick 4K Max (~$70)
- **Budget**: Chromecast with Google TV (~$50)
- **Media server**: Jellyfin (free) or Plex
</Aside>

## 🥇 Apple TV 4K

<Tabs>
  <TabItem label="Overview">
    ### Why Apple TV 4K?
    
    **Price:** ~$180
    
    Apple TV 4K is the **most polished** streaming experience:
    
    - ✅ Best UI and performance
    - ✅ No ads in interface
    - ✅ AirPlay for iPhone/Mac casting
    - ✅ HomeKit hub function
    - ✅ Apple Arcade gaming
    - ✅ Siri voice control
    - ✅ tvOS apps (Netflix, Disney+, etc.)
    
    **Disadvantages:**
    - ❌ Expensive
    - ❌ Requires Apple ecosystem for best experience
    - ❌ No Plex server function
  </TabItem>
  <TabItem label="Home Assistant">
    ### HA Integration
    
    Apple TV integrates via **Apple TV integration**:
    
    ```yaml
    # Auto-discovery - or manual addition
    # Settings → Devices → Add Integration → Apple TV
    
    # Entities:
    # - media_player.apple_tv_living_room
    # - remote.apple_tv_living_room
    
    # Example automation
    automation:
      - alias: "Dim lights during movie"
        trigger:
          - platform: state
            entity_id: media_player.apple_tv_living_room
            to: "playing"
        condition:
          - condition: template
            value_template: "{{ state_attr('media_player.apple_tv_living_room', 'media_content_type') == 'movie' }}"
        action:
          - service: light.turn_on
            entity_id: light.living_room
            data:
              brightness_pct: 10
    ```
  </TabItem>
</Tabs>

**Buy:** [Apple.com](https://apple.com), [Amazon](https://amazon.com), [Best Buy](https://bestbuy.com)

## 🥈 NVIDIA Shield TV Pro

<Tabs>
  <TabItem label="Overview">
    ### Why Shield TV Pro?
    
    **Price:** ~$200
    
    Shield TV Pro is the **power user's favorite**:
    
    - ✅ Built-in Plex Media Server
    - ✅ AI upscaling (HD → 4K)
    - ✅ Dolby Vision + Atmos
    - ✅ GeForce NOW cloud gaming
    - ✅ Kodi support
    - ✅ 2x USB 3.0 ports
    - ✅ 16GB storage + 3GB RAM
    - ✅ Android TV with all apps
    
    **Disadvantages:**
    - ❌ Expensive
    - ❌ Old hardware (2019)
    - ❌ Unknown when next generation comes
  </TabItem>
  <TabItem label="Plex Server">
    ### Shield as Plex Server
    
    Shield TV Pro can run **full Plex Media Server**:
    
    1. Install Plex Media Server from Play Store
    2. Connect external USB hard drive
    3. Add media libraries
    4. Stream to all devices in home
    
    **Advantages vs. NAS:**
    - Low power consumption (~15W)
    - Always on
    - Hardware transcoding
    - Simple setup
    
    **Limitations:**
    - Only 2 USB ports
    - Limited storage
  </TabItem>
  <TabItem label="Home Assistant">
    ### HA Integration
    
    Shield integrates via **Android TV integration**:
    
    ```yaml
    # Auto-discovery via Android TV
    # Settings → Devices → Add Integration → Android TV
    
    # Entities:
    # - media_player.shield_tv
    # - remote.shield_tv
    
    # Start Plex movie
    automation:
      - alias: "Movie night"
        trigger:
          - platform: event
            event_type: call_service
            event_data:
              domain: script
              service: movie_night
        action:
          - service: media_player.select_source
            target:
              entity_id: media_player.shield_tv
            data:
              source: "Plex"
    ```
  </TabItem>
</Tabs>

**Buy:** [Amazon](https://amazon.com), [Nvidia.com](https://nvidia.com)

## 🆓 Jellyfin - Free & Open Source

<Tabs>
  <TabItem label="Overview">
    ### Why Jellyfin?
    
    **Price:** 100% Free
    
    Jellyfin is the **best free alternative** to Plex:
    
    - ✅ Completely free - no premium tier
    - ✅ Open source
    - ✅ No tracking
    - ✅ Self-hosted (full control)
    - ✅ Hardware transcoding free
    - ✅ Live TV and DVR
    - ✅ Home Assistant integration
    
    **Disadvantages:**
    - ❌ Fewer polished client apps
    - ❌ Remote access requires more setup
    - ❌ Less automatic metadata
  </TabItem>
  <TabItem label="Installation">
    ### Docker Installation
    
    ```yaml
    # docker-compose.yml
    version: "3"
    services:
      jellyfin:
        image: jellyfin/jellyfin
        container_name: jellyfin
        network_mode: host
        volumes:
          - /path/to/config:/config
          - /path/to/cache:/cache
          - /path/to/media:/media
        restart: unless-stopped
        # Hardware acceleration (Intel)
        devices:
          - /dev/dri:/dev/dri
    ```
    
    ### Home Assistant Add-on
    
    Jellyfin is also available as a **HACS Add-on** for easy installation.
  </TabItem>
  <TabItem label="Home Assistant">
    ### HA Integration
    
    ```yaml
    # Settings → Devices → Add Integration → Jellyfin
    
    # Requires:
    # - Jellyfin server URL
    # - API key (Dashboard → API Keys)
    
    # Entities:
    # - media_player.jellyfin_living_room_tv
    # - sensor.jellyfin_watching
    
    # Automation example
    automation:
      - alias: "Pause Jellyfin on doorbell"
        trigger:
          - platform: state
            entity_id: binary_sensor.doorbell
            to: "on"
        condition:
          - condition: state
            entity_id: media_player.jellyfin_living_room_tv
            state: "playing"
        action:
          - service: media_player.media_pause
            entity_id: media_player.jellyfin_living_room_tv
    ```
  </TabItem>
</Tabs>

## 📀 Plex - Most Popular

<Tabs>
  <TabItem label="Overview">
    ### Why Plex?
    
    **Price:** Free / Plex Pass $120/year
    
    Plex is the **most popular** media server:
    
    - ✅ Best metadata and interface
    - ✅ Client apps for all platforms
    - ✅ Easy remote streaming
    - ✅ Plex Discover (streaming guide)
    - ✅ Large community
    
    **Plex Pass features:**
    - Hardware transcoding
    - Live TV & DVR
    - Offline sync
    - Intro skip
    
    **Disadvantages:**
    - ❌ Best features require payment
    - ❌ Cloud-dependent auth
    - ❌ View tracking
  </TabItem>
  <TabItem label="Home Assistant">
    ### HA Integration
    
    ```yaml
    # Plex auto-discovered
    # Settings → Devices → Add Integration → Plex
    
    # Entities per client:
    # - media_player.plex_living_room_tv
    # - media_player.plex_bedroom
    
    # Sensor for what's playing
    # - sensor.plex_watching
    
    # Automation: Movie lights
    automation:
      - alias: "Theater mode"
        trigger:
          - platform: state
            entity_id: media_player.plex_living_room
            to: "playing"
            attribute: media_content_type
        condition:
          - condition: template
            value_template: "{{ trigger.to_state.attributes.media_content_type == 'movie' }}"
        action:
          - service: scene.turn_on
            entity_id: scene.theater_mode
    ```
  </TabItem>
</Tabs>

## 🏠 Media Automations

### Basic: Lights on playback

```yaml
automation:
  - alias: "Dim lights on TV playback"
    trigger:
      - platform: state
        entity_id: media_player.living_room_tv
        to: "playing"
    action:
      - service: light.turn_on
        entity_id: light.living_room
        data:
          brightness_pct: 20
          transition: 5

  - alias: "Restore lights on pause/stop"
    trigger:
      - platform: state
        entity_id: media_player.living_room_tv
        from: "playing"
    action:
      - service: light.turn_on
        entity_id: light.living_room
        data:
          brightness_pct: 80
          transition: 3
```

### Advanced: Theater Scene

```yaml
# scenes.yaml
scene:
  - name: Theater Mode
    entities:
      light.living_room:
        state: on
        brightness_pct: 5
        color_temp: 500
      light.tv_backlight:
        state: on
        brightness_pct: 30
        rgb_color: [50, 50, 100]
      cover.living_room_blinds:
        state: closed

automation:
  - alias: "Activate theater on movie start"
    trigger:
      - platform: state
        entity_id: media_player.plex_living_room
        to: "playing"
    condition:
      - condition: template
        value_template: >
          {{ state_attr('media_player.plex_living_room', 'media_content_type') == 'movie' }}
      - condition: sun
        after: sunset
    action:
      - service: scene.turn_on
        entity_id: scene.theater_mode
```

### TTS Announcements

```yaml
automation:
  - alias: "Announcement on doorbell"
    trigger:
      - platform: state
        entity_id: binary_sensor.doorbell
        to: "on"
    action:
      # Pause music
      - service: media_player.media_pause
        entity_id: media_player.sonos_living_room
      # Announcement
      - service: tts.google_translate_say
        entity_id: media_player.sonos_living_room
        data:
          message: "Someone is at the door"
      # Wait and resume
      - delay: "00:00:05"
      - service: media_player.media_play
        entity_id: media_player.sonos_living_room
```
