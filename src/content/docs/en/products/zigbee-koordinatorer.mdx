A Zigbee coordinator is **the heart of your Zigbee network**. It connects all your Zigbee devices (sensors, bulbs, switches) to Home Assistant. Your choice of coordinator affects range, stability, and the number of devices you can connect.

<Aside type="tip" title="Our 2025 Recommendations">
- **Budget**: Sonoff ZBDongle-E (~$15) - Best price/quality
- **Best USB**: Sonoff Dongle Plus MG24 (~$35) - Latest chip
- **Best PoE/LAN**: SLZB-06 series (~$45-65) - Flexible placement
- **Premium**: Sonoff Dongle Max (~$55) - All-in-one solution
</Aside>

## ğŸ”Œ Connection Types

### USB Coordinators

Connect directly to your Home Assistant server via USB.

**Pros:**
- Cheaper
- Simple installation
- No extra network configuration

**Cons:**
- Must be close to server
- USB cable can cause interference
- Limited placement flexibility

### Ethernet/PoE Coordinators

Connect via network cable - can be placed anywhere.

**Pros:**
- Free placement in your home
- Better range (place centrally)
- PoE = only one cable
- Web dashboard for administration
- VPN support for remote locations

**Cons:**
- More expensive
- Requires PoE switch or injector
- More complex setup

## ğŸ“¶ Range Tips

1. **Use USB extension cable** - Away from USB 3.0 interference
2. **Place centrally** - PoE coordinators can be placed optimally
3. **Router devices** - Zigbee bulbs and plugs extend the network
4. **Avoid metal** - Don't place in metal enclosures
5. **Height** - Higher placement = better range
