## üèÜ Konnected Alarm Panel Pro (Recommended DIY)

<Tabs>
  <TabItem label="Overview">
    **Price:** ~$189 (6-zone), ~$249 (12-zone Pro)
    
    **Works with Home Assistant** certified! ‚úÖ
    
    Reuse your **existing wired sensors** from an old alarm system:
    
    - ‚úÖ ESP32-based with ESPHome firmware
    - ‚úÖ WiFi, Ethernet and PoE support
    - ‚úÖ 6-12 zones for sensors
    - ‚úÖ Siren and strobe outputs
    - ‚úÖ Local control, no cloud
    - ‚úÖ No monthly fees
    - ‚úÖ Standalone alarm mode (Nov 2024)
    
    <Aside type="note">
    Konnected supports standard wired sensors: door contacts, PIR motion, glass break sensors, smoke detectors, etc.
    </Aside>
  </TabItem>
  <TabItem label="Installation">
    ### ESPHome Firmware
    
    Konnected now uses **ESPHome** (legacy integration is deprecated):
    
    1. Go to [install.konnected.io](https://install.konnected.io)
    2. Select your device and firmware
    3. Connect via USB and flash
    4. Connect to WiFi via captive portal
    5. Automatically discovered in Home Assistant
    
    ### Zone Configuration
    
    ```yaml
    # ESPHome configuration (automatic from Konnected)
    # Customize in ESPHome Dashboard
    
    binary_sensor:
      - platform: gpio
        pin:
          number: $zone1
          mode: INPUT_PULLUP
        name: "Front Door"
        device_class: door
        
      - platform: gpio
        pin:
          number: $zone2
          mode: INPUT_PULLUP
        name: "Living Room Motion"
        device_class: motion
    
    switch:
      - platform: gpio
        pin: $alarm
        name: "Siren"
        icon: "mdi:alarm-light"
    ```
  </TabItem>
  <TabItem label="Automations">
    ### Konnected + Alarmo Integration
    
    ```yaml
    # Alarmo uses Konnected sensors automatically
    # Set up in Alarmo UI:
    # 1. Add Konnected binary_sensors
    # 2. Assign to zones (Entry, Interior)
    # 3. Configure delays
    
    automation:
      # Activate siren when alarm triggers
      - alias: "Alarm - Activate Konnected Siren"
        trigger:
          - platform: state
            entity_id: alarm_control_panel.alarmo
            to: "triggered"
        action:
          - service: switch.turn_on
            entity_id: switch.konnected_siren
          - delay: "00:03:00"  # Siren for 3 min
          - service: switch.turn_off
            entity_id: switch.konnected_siren
    
      # Stop siren on disarm
      - alias: "Alarm - Stop Siren on Disarm"
        trigger:
          - platform: state
            entity_id: alarm_control_panel.alarmo
            to: "disarmed"
        action:
          - service: switch.turn_off
            entity_id: switch.konnected_siren
    ```
  </TabItem>
</Tabs>

**Buy:** [Konnected.io](https://konnected.io)

## üõ°Ô∏è Alarmo (HACS - Recommended)

<Tabs>
  <TabItem label="Overview">
    **Price:** FREE (open source)
    
    **The best way** to build an alarm system in Home Assistant:
    
    - ‚úÖ Complete UI-based setup
    - ‚úÖ No YAML required
    - ‚úÖ 4 modes: Home, Away, Night, Custom
    - ‚úÖ User codes with different permissions
    - ‚úÖ Entry/Exit delays
    - ‚úÖ Push notifications
    - ‚úÖ Zones and sensor groups
    - ‚úÖ Beautiful Lovelace alarm panel card
    
    <Aside type="tip">
    Alarmo uses your existing sensors - Zigbee, Z-Wave, WiFi, ESPHome, anything!
    </Aside>
  </TabItem>
  <TabItem label="Installation">
    ### HACS Installation
    
    1. Open HACS ‚Üí Integrations
    2. Click "+" and search "Alarmo"
    3. Download and restart Home Assistant
    4. Go to Settings ‚Üí Devices ‚Üí Add Integration
    5. Search "Alarmo" and add
    6. Alarmo menu appears in sidebar
    
    ### Basic Configuration
    
    In Alarmo panel:
    
    **General tab:**
    - Enable modes (Away, Home, Night)
    - Set Exit delay (time to leave)
    - Set Entry delay (time to disarm)
    - Trigger time (siren duration)
    
    **Sensors tab:**
    - Add door/window sensors
    - Add motion sensors
    - Configure per-sensor delays
  </TabItem>
  <TabItem label="Configuration">
    ### Sensor Setup
    
    For each sensor, configure:
    
    - **Immediate:** Triggers alarm instantly (windows)
    - **Entry delay:** Gives time to disarm (front door)
    - **Exit delay:** Ignored during arming
    - **Bypass:** Can be skipped when arming
    
    ### User Codes
    
    ```yaml
    # Alarmo UI - Codes tab
    # Create different users:
    
    # Admin - full access
    # User: "Mom and Dad"
    # Code: 1234
    # Permissions: Arm + Disarm
    
    # Limited - disarm only
    # User: "Kids"
    # Code: 5678
    # Permissions: Disarm only
    
    # Cleaning - time limited
    # User: "Cleaner"
    # Code: 9999
    # Permissions: Arm + Disarm
    # Schedule: Tuesdays 10:00-14:00
    ```
    
    ### Actions (Automations)
    
    Alarmo has built-in actions system:
    
    **Notifications:**
    - Alarm armed ‚Üí Push to all
    - Alarm triggered ‚Üí Push + SMS
    - Failed to arm ‚Üí Show which sensor
    
    **Devices:**
    - Triggered ‚Üí Turn on all lights
    - Triggered ‚Üí Activate siren
    - Triggered ‚Üí Record cameras
  </TabItem>
  <TabItem label="Dashboard">
    ### Alarmo Card (HACS)
    
    ```yaml
    # HACS ‚Üí Frontend ‚Üí Search "Alarmo Card"
    
    # Dashboard configuration
    type: custom:alarmo-card
    entity: alarm_control_panel.alarmo
    show_keypad: true
    button_scale: 1.2
    states:
      - arm_away
      - arm_home
      - arm_night
    ```
    
    ### Complete Alarm Dashboard
    
    ```yaml
    type: vertical-stack
    cards:
      # Alarmo control
      - type: custom:alarmo-card
        entity: alarm_control_panel.alarmo
        show_keypad: true
      
      # Sensor status
      - type: entities
        title: "Security Sensors"
        entities:
          - entity: binary_sensor.front_door
            name: "Front Door"
          - entity: binary_sensor.back_door
            name: "Back Door"
          - entity: binary_sensor.living_room_motion
            name: "Living Room Motion"
          - entity: binary_sensor.garage_door
            name: "Garage Door"
      
      # Quick arm/disarm
      - type: horizontal-stack
        cards:
          - type: button
            name: "Away"
            icon: mdi:shield-lock
            tap_action:
              action: call-service
              service: alarmo.arm
              service_data:
                entity_id: alarm_control_panel.alarmo
                mode: away
                code: !secret alarm_code
          - type: button
            name: "Home"
            icon: mdi:shield-home
            tap_action:
              action: call-service
              service: alarmo.arm
              service_data:
                entity_id: alarm_control_panel.alarmo
                mode: home
    ```
  </TabItem>
</Tabs>

    ```yaml
      # Arm when everyone leaves

          - platform: state

          - service: alarm_control_panel.alarm_arm_away

          - service: alarm_control_panel.alarm_disarm

      - alias: "SimpliSafe - Doorbell"
        trigger:
          - platform: event
            event_type: SIMPLISAFE_EVENT
            event_data:
              last_event_type: doorbell_detected
    ```

```yaml
automation:
    ```

```yaml
  - alias: "Alarm - Night Arm"

      - platform: time
        at: "23:00:00"
    condition:
      - condition: state
        entity_id: group.family
        state: "home"
      - condition: state
        entity_id: alarm_control_panel.alarmo
        state: "disarmed"
    action:
      - service: alarmo.arm
        data:
          entity_id: alarm_control_panel.alarmo
          mode: night

  # Disarm in the morning
  - alias: "Alarm - Morning Disarm"
    trigger:
      - platform: time
        at: "06:30:00"
    condition:
      - condition: state
        entity_id: alarm_control_panel.alarmo
        state: "armed_night"
    action:
      - service: alarmo.disarm
        data:
          entity_id: alarm_control_panel.alarmo

  # ===== TRIGGERED RESPONSE =====
  
  # Turn on all lights
  - alias: "Alarm - Triggered Lights"
    trigger:
      - platform: state
        entity_id: alarm_control_panel.alarmo
        to: "triggered"
    action:
      - service: light.turn_on
        entity_id: all
        data:
          brightness_pct: 100
      - service: switch.turn_on
        entity_id: switch.outdoor_lights

  # Push notification with actionable buttons
  - alias: "Alarm - Triggered Notification"
    trigger:
      - platform: state
        entity_id: alarm_control_panel.alarmo
        to: "triggered"
    action:
      - service: notify.mobile_app_phone
        data:
          title: "üö® ALARM TRIGGERED!"
          message: >
            Sensor: {{ state_attr('alarm_control_panel.alarmo', 'open_sensors') }}
            Time: {{ now().strftime('%H:%M') }}

            actions:
              - action: "DISARM_ALARM"
                title: "Disarm"

            entity_id: alarm_control_panel.alarmo

      - service: alarmo.disarm
```

## üìã Sensor Types Guide

### Which Sensors for Which Zones

| Sensor Type | Entry Zone | Interior | Perimeter | 24/7 |
|-------------|-----------|----------|-----------|------|
| Front Door | ‚úÖ | | | |
| Back Door | ‚úÖ | | | |
| Windows | | | ‚úÖ | |
| Living Room Motion | | ‚úÖ | | |
| Hallway Motion | ‚úÖ | | | |
| Glass Break | | | ‚úÖ | |
| Smoke Detector | | | | ‚úÖ |
| CO Detector | | | | ‚úÖ |
| Water Leak | | | | ‚úÖ |

### Delay Recommendations

```yaml
# Entry Zone (front door, back door)
entry_delay: 30  # seconds to disarm
exit_delay: 60   # seconds to leave

# Interior (motion)
entry_delay: 0   # triggers immediately
exit_delay: 60   # ignored during exit

# Perimeter (windows, glass break)
entry_delay: 0   # triggers immediately
exit_delay: 0    # triggers immediately

# 24/7 (smoke, CO, water)
always_on: true  # active regardless of arm status
```
