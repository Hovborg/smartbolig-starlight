## üîë Smart Bulbs vs Smart Switches

### What should you choose?

| Smart Bulbs | Smart Switches/Dimmers |
|-------------|------------------------|
| ‚úÖ Colors and CCT (warm/cool white) | ‚úÖ Keep your existing bulbs |
| ‚úÖ Individual control per bulb | ‚úÖ Always works physically |
| ‚úÖ Easy installation | ‚úÖ Cheaper in the long run |
| ‚ùå Doesn't work if switch is off | ‚ùå No colors |
| ‚ùå More expensive per light point | ‚ùå Requires electrician for installation |

<Aside type="note" title="Our Recommendation">
- **Living rooms and bedrooms**: Smart bulbs for mood and colors
- **Kitchen, bathroom, hallway**: Smart switches/dimmers
- **Outdoor**: Smart switches (weatherproof bulbs are rare)
</Aside>

### ü•á WLED - Best DIY Solution

WLED is **open source firmware** for ESP32/ESP8266 that controls addressable LED strips (WS2812B, SK6812, etc.).

**Advantages:**
- ‚úÖ Amazing effect library (100+ effects)
- ‚úÖ Native Home Assistant integration
- ‚úÖ Segment control
- ‚úÖ Sync between multiple strips
- ‚úÖ Extremely cheap

**What you need:**
- ESP32 or ESP8266 (~$4)
- WS2812B LED strip (~$4-7/meter)
- 5V power supply (2A per meter)

**Buy:** [AliExpress](https://aliexpress.com) (search "WS2812B 5V")

## üè† Home Assistant Automations

### Popular Lighting Automations

```yaml
# Turn on light on motion
automation:
  - alias: "Light on motion"
    trigger:
      - platform: state
        entity_id: binary_sensor.hallway_motion
        to: "on"
    condition:
      - condition: sun
        after: sunset
    action:
      - service: light.turn_on
        target:
          entity_id: light.hallway
        data:
          brightness_pct: 100

# Gradual dimming in evening
  - alias: "Evening lighting"
    trigger:
      - platform: sun
        event: sunset
        offset: "+01:00:00"
    action:
      - service: light.turn_on
        target:
          entity_id: light.living_room
        data:
          color_temp_kelvin: 2700
          brightness_pct: 50
```

### Adaptive Lighting

Home Assistant has an **Adaptive Lighting** integration that automatically adjusts:
- Color temperature based on time of day
- Brightness based on sunlight
- Gradual wake-up and sleep modes

**Installation:**
1. HACS ‚Üí Search "Adaptive Lighting"
2. Install and restart
3. Configure which lights to control
