Nu begynder det sjove! Automationer er grunden til at vi gider alt det her. At lyset tænder når du kommer hjem. At varmen skruer ned når du går.

## 🧠 Sådan virker automationer

Alle automationer har tre dele:

![Automation flow diagram](/images/guides/foerste-automation/flow.svg)

**Tænk på det som:** "Når X sker, og hvis Y er sandt, så gør Z."

## 🌅 Eksempel 1: Lys ved solnedgang

Den klassiske første automation!

<Tabs>
  <TabItem label="📱 Via UI (nemmest)">
    <Steps>
    1. Gå til **Indstillinger** → **Automatiseringer og scener**
    
    2. Klik **Opret automatisering** → **Opret ny automatisering**
    
    3. **Trigger:** 
       - Klik **"Tilføj trigger"**
       - Vælg **"Sol"**
       - Vælg **"Solnedgang"**
    
    4. **Handling:**
       - Klik **"Tilføj handling"**
       - Vælg **"Enhed"**
       - Find din lampe
       - Vælg **"Tænd"**
    
    5. **Gem** med navnet "Tænd stuelys ved solnedgang"
    </Steps>
  </TabItem>
  <TabItem label="📝 YAML">
    ```yaml title="automations.yaml"
    - alias: "Tænd stuelys ved solnedgang"
      description: "Tænder automatisk lyset når solen går ned"
      trigger:
        - platform: sun
          event: sunset
          offset: "-00:30:00"  # 30 min FØR solnedgang
      action:
        - service: light.turn_on
          target:
            entity_id: light.stue_loft
          data:
            brightness_pct: 80
      mode: single
    ```
  </TabItem>
</Tabs>

## 🚶 Eksempel 2: Lys ved bevægelse (kun om natten)

```yaml title="automations.yaml"
- alias: "Gang lys ved bevægelse (nat)"
  description: "Tænder svagt lys i gangen om natten"
  trigger:
    - platform: state
      entity_id: binary_sensor.gang_bevaegelse
      to: "on"
  condition:
    - condition: time
      after: "22:00:00"
      before: "07:00:00"
  action:
    - service: light.turn_on
      target:
        entity_id: light.gang
      data:
        brightness_pct: 20  # Svagt lys om natten
    - delay:
        minutes: 2
    - service: light.turn_off
      target:
        entity_id: light.gang
  mode: restart  # Genstarter timer ved ny bevægelse
```

<Aside type="tip" title="Hvad gør mode: restart?">
Hvis du bevæger dig igen inden de 2 minutter, starter timeren forfra. Ellers ville lyset slukke mens du stadig er i gangen!
</Aside>

## 🧺 Eksempel 3: Vaskemaskine færdig

Kræver smart stikkontakt med strømmåling (f.eks. IKEA INSPELNING eller Sonoff S31):

```yaml title="automations.yaml"
- alias: "Vaskemaskine færdig"
  description: "Sender besked når vasketøjet er færdigt"
  trigger:
    - platform: numeric_state
      entity_id: sensor.vaskemaskine_stromforbrug
      below: 5  # Watt
      for:
        minutes: 3  # Skal være lav i 3 min
  condition:
    - condition: numeric_state
      entity_id: sensor.vaskemaskine_stromforbrug
      above: 0.5  # Sikrer at den har kørt
  action:
    - service: notify.mobile_app_min_telefon
      data:
        title: "🧺 Vaskemaskine"
        message: "Vasketøjet er færdigt!"
  mode: single
```

## 🌡️ Eksempel 4: Temperatur advarsel

```yaml title="automations.yaml"
- alias: "Lav temperatur advarsel"
  description: "Advarer hvis det bliver for koldt"
  trigger:
    - platform: numeric_state
      entity_id: sensor.stue_temperatur
      below: 18
      for:
        minutes: 10
  action:
    - service: notify.mobile_app_min_telefon
      data:
        title: "🥶 Koldt i stuen"
        message: "Temperaturen er faldet til {{ states('sensor.stue_temperatur') }}°C"
  mode: single
```

## 🔧 Debugging

<Aside type="tip" title="Automation virker ikke?">
**Tjek disse ting:**

1. **Entity ID** - Stavefejl er den hyppigste fejl!
   - Find korrekt ID under **Udviklerværktøjer** → **Tilstande**

2. **Logs** - Se fejlbeskeder
   - **Indstillinger** → **System** → **Logs**

3. **Sporing** - Se seneste kørsler
   - Klik på automationen → **Spor** (nederst)

4. **Test trigger manuelt**
   - **Udviklerværktøjer** → **Tilstande** → Ændr entity tilstand
</Aside>

## 📚 Nyttige trigger-typer

| Trigger | Hvornår | Eksempel |
|---------|---------|----------|
| `state` | Entity ændrer tilstand | Bevægelse, dør åbner |
| `numeric_state` | Værdi over/under | Temperatur, strømforbrug |
| `sun` | Solopgang/nedgang | Lys automation |
| `time` | Bestemt klokkeslæt | Morgenrutine |
| `zone` | Person ankommer/forlader | Hjemme/væk |
| `device` | Enhedsspecifik | Knap trykkes |
