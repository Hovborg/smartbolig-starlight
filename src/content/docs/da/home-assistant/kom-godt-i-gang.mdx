Home Assistant er en af de mest brugte open source-platforme til hjemmeautomatisering. I denne guide får du et komplet overblik over hvad Home Assistant er, og hvordan du kommer i gang.

## 🤔 Hvad er Home Assistant?

![Home Assistant oversigt](/images/guides/kom-godt-i-gang-ha/overview.png)

Home Assistant er **gratis software** der samler alle dine smart home enheder i én app. Tænk på det som "hjernen" i dit smarte hjem.

### Hvad kan det?

- 🔌 **Forbinde over 2.000 integrationer** - IKEA, Philips Hue, Sonos og mange flere
- 🤖 **Lave automationer** - "Tænd lys når solen går ned"
- 📊 **Vise data** - Grafer over strømforbrug, temperatur, osv.
- 🔒 **Holde data lokalt** - Ingen cloud, ingen sporing

### Arkitektur oversigt

![Home Assistant arkitektur](/images/guides/kom-godt-i-gang-ha/arkitektur.svg)

## 💻 Hvad har du brug for?

Du kan køre Home Assistant på mange forskellige måder. Her er de tre mest populære:

| Platform | Pris ca. | Sværhedsgrad | Bedst til |
|----------|----------|--------------|-----------|
| **Raspberry Pi** | ~650 kr | Let | Begyndere |
| **Proxmox VM** | Gratis* | Medium | Eksisterende server |
| **Docker** | Gratis* | Avanceret | Erfarne brugere |

*Kræver eksisterende hardware

## 🚀 Vælg din installationsmetode

### 1. Raspberry Pi (anbefalet for begyndere)

**Hvad:** Home Assistant OS på en dedikeret Raspberry Pi.

| Fordele | Ulemper |
|---------|---------|
| ✅ Alt virker out-of-the-box | ❌ Kræver dedikeret hardware |
| ✅ Nem opdatering med ét klik | ❌ Mindre kraftfuld end PC |
| ✅ Fuld add-on support | ❌ SD-kort kan slides |
| ✅ Lavt strømforbrug (~5W) | |

**Hardware du skal bruge:**

| Del | Pris ca. |
|-----|----------|
| Raspberry Pi 4/5 (4GB) | 450 kr |
| Officiel strømforsyning | 100 kr |
| SD-kort 32GB+ eller SSD | 100 kr |
| **Total** | **~650 kr** |

### 2. Proxmox VM (anbefalet for hjemmeservere)

**Hvad:** Home Assistant OS i en virtuel maskine på Proxmox VE.

| Fordele | Ulemper |
|---------|---------|
| ✅ Snapshots - rul tilbage med ét klik | ❌ Kræver Proxmox kendskab |
| ✅ Fuld add-on support | ❌ USB passthrough kan være tricky |
| ✅ Kør andre VMs/containers samtidig | |
| ✅ Enterprise backup og restore | |
| ✅ Nem skalering af ressourcer | |

**Krav:**
- Proxmox VE 8.x+ installeret
- 2+ CPU kerner ledige
- 4+ GB RAM ledigt
- 32+ GB storage

<Card title="🖥️ Installer på Proxmox" icon="server">
  Perfekt hvis du har en hjemmeserver med Proxmox.
  
  [Følg Proxmox guiden →](/da/home-assistant/proxmox-installation/)
</Card>

### 3. Docker Container (for erfarne)

**Hvad:** Home Assistant Container på eksisterende Linux server eller NAS.

| Fordele | Ulemper |
|---------|---------|
| ✅ Kører sammen med anden software | ❌ Ingen Supervisor/Add-ons |
| ✅ Fuld kontrol over systemet | ❌ Kræver Docker kendskab |
| ✅ Lavt ressourceforbrug | ❌ Manuel opsætning af MQTT osv. |
| ✅ Perfekt til NAS (Synology, QNAP) | |

## 📊 Sammenligning af installationsmetoder

| Feature | Raspberry Pi | Proxmox VM | Docker |
|---------|-------------|------------|--------|
| **Sværhedsgrad** | ⭐ Let | ⭐⭐ Medium | ⭐⭐⭐ Avanceret |
| **Supervisor/Add-ons** | ✅ Fuld | ✅ Fuld | ❌ Nej |
| **Snapshots** | ❌ Nej | ✅ Ja | ❌ Nej |
| **Ressourceforbrug** | Lavt | Medium | Lavt |
| **Opdateringer** | Auto | Auto | Manuel |
| **Backup** | Via HA | Via Proxmox | Manuel |
| **Andre services** | ❌ Nej | ✅ Ja | ✅ Ja |
| **USB enheder** | ✅ Direkte | ⚠️ Passthrough | ⚠️ Mount |

<Aside type="tip" title="Ikke sikker på hvad du skal vælge?">
- **Helt ny til smart home?** → Raspberry Pi
- **Har du en Proxmox server?** → Proxmox VM
- **Har du en NAS eller Docker server?** → Docker
- **Vil du bare prøve det?** → Raspberry Pi
</Aside>
