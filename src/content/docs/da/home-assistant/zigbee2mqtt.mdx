![Zigbee2MQTT](/images/guides/zigbee2mqtt-hero.png)

Zigbee er fantastisk til smart home - lavt strømforbrug, mesh-netværk, og enheder der holder i årevis på ét batteri. Zigbee2MQTT giver dig én USB-dongle til **alle** Zigbee enheder - IKEA, Aqara, Sonoff, Philips Hue, alt sammen.

<Aside type="note" title="Open Source">
Zigbee2MQTT er open source! Se [Zigbee2MQTT på GitHub](https://github.com/Koenkk/zigbee2mqtt) for dokumentation og [understøttede enheder](https://www.zigbee2mqtt.io/supported-devices/).
</Aside>

## 🤔 Hvorfor Zigbee2MQTT?

| Feature | Zigbee2MQTT | Producent-hub (IKEA/Hue) |
|---------|-------------|--------------------------|
| Pris | ~200 kr (én gang) | 300-500 kr per hub |
| Enheder | 2800+ fra alle mærker | Kun eget mærke |
| Lokal | ✅ 100% lokal | ⚠️ Ofte cloud |
| Åben | ✅ Open source | ❌ Lukket |

Brug ALTID en USB-forlænger (1-2m). WiFi og Zigbee interfererer - afstand hjælper!
</Aside>

## ⚙️ Installation

<Tabs>
  <TabItem label="Home Assistant OS">
    <Steps>
    1. **Installer Mosquitto MQTT broker**
       
       Gå til **Indstillinger** → **Add-ons** → **Add-on butik**
       
       Søg **"Mosquitto broker"** → **Installer** → **Start**

    2. **Opret MQTT bruger**
       
       **Indstillinger** → **Personer** → **Brugere** → **Tilføj bruger**
       
       - Brugernavn: `mqtt`
       - Password: (vælg noget sikkert)
       - Administrator: ❌ Nej

    3. **Tilføj Zigbee2MQTT repository**
       
       **Indstillinger** → **Add-ons** → **Add-on butik** → **⋮** (tre prikker) → **Repositories**
       
       Tilføj:
       ```
       https://github.com/zigbee2mqtt/hassio-zigbee2mqtt
       ```

    4. **Installer Zigbee2MQTT**
       
       Søg **"Zigbee2MQTT"** → **Installer** (brug edge-versionen for nyeste features)

    5. **Find din USB-dongle**
       
       **Indstillinger** → **System** → **Hardware** → **All Hardware**
       
       Find stien der ligner:
       ```
       /dev/serial/by-id/usb-Silicon_Labs_Sonoff_Zigbee_3.0_USB_Dongle_Plus-if00-port0
       ```

    6. **Konfigurer Zigbee2MQTT**
       
       Klik på Zigbee2MQTT add-on → **Konfiguration**:
       ```yaml
       serial:
         port: /dev/serial/by-id/usb-Silicon_Labs_Sonoff...
       mqtt:
         server: mqtt://core-mosquitto
         user: mqtt
         password: dit-mqtt-password
       frontend:
         port: 8080
       ```

    7. **Start og åbn**
       
       Klik **Start** og åbn: `http://homeassistant.local:8080`
    </Steps>
  </TabItem>
  <TabItem label="Docker">
    ```yaml title="docker-compose.yml"
    services:
      mosquitto:
        container_name: mosquitto
        image: eclipse-mosquitto
        volumes:
          - ./mosquitto/config:/mosquitto/config
          - ./mosquitto/data:/mosquitto/data
        ports:
          - "1883:1883"
        restart: unless-stopped

      zigbee2mqtt:
        container_name: zigbee2mqtt
        image: koenkk/zigbee2mqtt
        volumes:
          - ./zigbee2mqtt/data:/app/data
          - /run/udev:/run/udev:ro
        devices:
          - /dev/ttyUSB0:/dev/ttyUSB0
        ports:
          - "8080:8080"
        environment:
          - TZ=Europe/Copenhagen
        restart: unless-stopped
    ```
  </TabItem>
</Tabs>

## 📡 Tilføj enheder

<Steps>
1. Åbn Zigbee2MQTT web-interface

2. Klik **"Permit join (All)"** øverst

3. Par enheden (se tabel nedenfor)

4. Enheden dukker op - giv den et beskrivende navn

5. Klik **"Disable join"** når du er færdig
</Steps>

### Parringsinstruktioner

| Enhed | Sådan parrer du |
|-------|-----------------|
| **Aqara sensorer** | Hold knap 5 sek indtil LED blinker |
| **IKEA pære** | Tænd/sluk 6 gange hurtigt |
| **IKEA fjernbetjening** | Hold parknap 10 sek ved batteriet |
| **Sonoff kontakt** | Hold knap 5 sek |
| **Philips Hue** | Tænd/sluk 5 gange (eller brug Hue Bluetooth app) |

<Aside type="note">
Enheden skal være tæt på koordinatoren ved parring (~2m). Bagefter kan du flytte den.
</Aside>

## 🌐 Mesh-netværk tips

![Zigbee mesh netværk diagram](/images/guides/zigbee2mqtt/mesh.svg)

Zigbee er et mesh-netværk. Enheder med strøm (routere) forstærker signalet.

### Routere vs End devices

| Type | Beskrivelse | Eksempler |
|------|-------------|-----------|
| **Koordinator** | Din USB-dongle | Sonoff Dongle |
| **Router** | Forstærker signal | Pærer, stikkontakter |
| **End device** | Kun modtager | Sensorer (på batteri) |

<Aside type="tip" title="Pro tip: Smarte pærer som repeatere">
Placer smarte pærer/stikkontakter strategisk i hjemmet. De fungerer som repeatere og udvider dit netværk!
</Aside>

### WiFi interferens

Zigbee og WiFi bruger begge 2.4 GHz. Undgå problemer:

- Sæt WiFi på kanal **1** eller **11**
- Lad Zigbee2MQTT vælge kanal automatisk (default 11)
- Brug USB-forlænger til koordinatoren

## 🔧 Fejlfinding

<Aside type="tip" title="Enhed parrer ikke?">
1. Er "Permit join" aktiveret?
2. Nulstil enheden (se manual)
3. Flyt tættere på koordinatoren
4. Tjek at koordinatoren ikke er for tæt på WiFi router
</Aside>

<Aside type="tip" title="Enhed falder fra netværket?">
1. Dårligt signal - tilføj flere routere (smarte pærer/stikkontakter)
2. Tjek batteri på sensorer
3. Prøv at gen-parre enheden
</Aside>

<Aside type="tip" title="Zigbee2MQTT starter ikke?">
1. Tjek at USB-stien er korrekt
2. Tjek MQTT brugernavn/password
3. Se logs i add-on for fejlbeskeder
</Aside>
