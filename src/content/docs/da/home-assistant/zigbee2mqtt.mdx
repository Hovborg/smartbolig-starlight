Zigbee er fantastisk. Lavt strømforbrug, mesh-netværk, og enheder der bare virker. Zigbee2MQTT giver dig én USB-dongle til alle enheder - Aqara, IKEA, Sonoff, Philips Hue, alt sammen.

## Hvad skal du bruge?

**Zigbee USB-dongle:** <Badge text="Anbefalet" variant="success" />
- Sonoff Zigbee 3.0 USB Dongle Plus (150-200 kr)
- SLZB-06 (Ethernet-baseret, dyrere men stabil)

**MQTT Broker:**
- Mosquitto (standard, gratis)

## Installation

<Aside type="note">
Denne guide er til Home Assistant OS. Bruger du Docker? Kør Zigbee2MQTT som separat container.
</Aside>

<Steps>
1. **Installer Mosquitto broker**
   
   Gå til **Indstillinger** → **Add-ons** → **Add-on butik** → Søg "Mosquitto broker" → **Installer** og **Start**

2. **Opret MQTT bruger**
   
   **Indstillinger** → **Personer** → **Brugere** → Opret ny:
   - Brugernavn: `mqtt`
   - Password: (vælg noget sikkert)
   - Kun lokal adgang: ✅

3. **Installer Zigbee2MQTT**
   
   Add-on butik → Tre prikker → **Repositories** → Tilføj:
   ```
   https://github.com/zigbee2mqtt/hassio-zigbee2mqtt
   ```
   Søg "Zigbee2MQTT" og installer.

4. **Find din dongle**
   
   **Indstillinger** → **System** → **Hardware** → **All Hardware**
   
   Led efter noget som:
   ```
   /dev/serial/by-id/usb-Silicon_Labs_Sonoff_Zigbee_3.0_USB_Dongle_Plus-if00-port0
   ```

5. **Konfigurer Zigbee2MQTT**
   
   ```yaml title="Zigbee2MQTT konfiguration"
   serial:
     port: /dev/serial/by-id/usb-Silicon_Labs_Sonoff...
   mqtt:
     server: mqtt://core-mosquitto
     user: mqtt
     password: dit-mqtt-password
   frontend:
     port: 8485
   ```

6. **Start Zigbee2MQTT**
   
   Åbn web-interface på `http://homeassistant.local:8485`
</Steps>

## Tilføj enheder

<Steps>
1. Klik **Permit join** i Zigbee2MQTT interfacet
2. Par enheden (se tabel nedenfor)
3. Giv enheden et fornuftigt navn
</Steps>

| Enhed | Parringsmetode |
|-------|----------------|
| Aqara sensor | Hold knap 5 sek til LED blinker |
| IKEA pære | Tænd/sluk 6 gange hurtigt |
| Sonoff kontakt | Hold knap 5 sekunder |

## Netværkstips

<Aside type="tip" title="Placering af koordinator">
- Minimum 1 meter fra router/WiFi
- Ikke bag metal eller beton
- USB-forlænger kan hjælpe med rækkevidde
</Aside>

### Mesh-netværk

Zigbee er mesh - enheder sender gennem hinanden. Men kun enheder med strøm (routere) gør det. **Placer smarte stikkontakter og pærer strategisk** - de forstærker signalet.

### Zigbee vs WiFi

Begge bruger 2.4 GHz. Tommelfingerregel: Sæt WiFi på kanal 1 eller 6, lad Zigbee2MQTT vælge automatisk.

## Populære enheder

**Sensorer:**
- Aqara dør/vindue sensor - billig og pålidelig
- Aqara temperatur/fugtighed - batteriet holder evigt

**Kontakter:**
- Sonoff ZBMINI - passer bag kontakten
- IKEA TRETAKT - billig smart stikkontakt

**Pærer:**
- IKEA TRÅDFRI - godt lys, fin pris
- Philips Hue - virker uden Hue hub

## Fejlfinding

| Problem | Løsning |
|---------|---------|
| Enhed parrer ikke | Er "Permit join" aktiveret? Nulstil enheden |
| Enhed falder fra | Dårlig signal, tilføj flere routere |
| Z2M starter ikke | Check port-sti og MQTT login |
