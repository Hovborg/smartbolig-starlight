Home Assistant er verdens mest populære open source smart home platform. Saml alle dine enheder i én app - helt uden cloud.

<Aside type="note" title="Open Source">
Home Assistant er open source! Se [Home Assistant på GitHub](https://github.com/home-assistant/core) for udvikling og [HACS](https://github.com/hacs/integration) for community integrationer.
</Aside>

### Hvilken metode skal jeg vælge?

| Metode | Bedst til | Sværhedsgrad | Add-ons |
|--------|-----------|--------------|---------|
| **Raspberry Pi** | Begyndere, dedikeret enhed | ⭐ Let | ✅ Ja |
| **Proxmox VM** | Hjemmeserver, homelab | ⭐⭐ Medium | ✅ Ja |
| **Docker** | NAS, eksisterende server | ⭐⭐⭐ Avanceret | ❌ Nej |

<Aside type="tip" title="Ikke sikker?">
- **Helt ny?** → Raspberry Pi
- **Har Proxmox server?** → Proxmox VM
- **Har NAS/Docker?** → Docker
</Aside>
