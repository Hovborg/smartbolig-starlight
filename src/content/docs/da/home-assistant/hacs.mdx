**Home Assistant Community Store (HACS)** er en custom integration, der giver dig adgang til tusindvis af community-udviklede integrationer, themes og Lovelace kort. HACS er næsten et must-have for alle Home Assistant brugere.

<Aside type="tip">
HACS er ikke en officiel Home Assistant add-on, men et community-projekt der kører direkte i Home Assistant. Lav altid backup før installation!
</Aside>

## Hvad er HACS?

HACS fungerer som en "app store" for Home Assistant, hvor du kan:

- **Opdage** nye custom integrationer og komponenter
- **Installere** med få klik direkte fra brugerfladen  
- **Opdatere** alt fra ét sted
- **Administrere** dine installerede tilføjelser

  title="HACS på GitHub"

## Forudsætninger

Før du starter, skal du have:

- ✅ Home Assistant kørende (OS, Supervised, Container eller Core)
- ✅ En gratis [GitHub konto](https://github.com/signup)
- ✅ Adgang til Home Assistant webinterface
- ✅ En backup af dit system

<Aside type="caution">
HACS installerer custom komponenter der kører direkte i Home Assistant. Der er altid en lille risiko for at noget kan gå galt - lav derfor altid backup først!
</Aside>

### Container/Core Installation

Hvis du kører Home Assistant Container eller Core, skal du køre dette script:

```bash
# Kør i din Home Assistant config mappe
wget -O - https://get.hacs.xyz | bash -
```

Genstart derefter Home Assistant og fortsæt fra trin 6 ovenfor.

## Installer din første integration

Lad os installere en populær integration som eksempel:

<Steps>
1. Klik på **HACS** i sidemenuen
2. Vælg **Integrationer**
3. Klik på **+ Udforsk og download repositories** (nederst til højre)
4. Søg efter f.eks. "Energi Data Service" (dansk elpris-integration)
5. Klik på integrationen og læs beskrivelsen
6. Klik **Download** → vælg version → **Download**
7. **Genstart Home Assistant** når du bliver bedt om det
8. Tilføj integrationen via Indstillinger → Enheder og tjenester → Tilføj integration
</Steps>

<Aside type="tip">
Efter download af en integration skal du altid genstarte Home Assistant og derefter tilføje integrationen under Enheder og tjenester.
</Aside>

## Opdatering af HACS komponenter

HACS gør det nemt at holde alt opdateret:

<Steps>
1. Gå til **HACS** i sidemenuen
2. Se efter et **orange badge** der viser tilgængelige opdateringer
3. Klik på komponenten med opdatering
4. Klik **Opdater** og følg instruktionerne
5. Genstart Home Assistant hvis påkrævet
</Steps>

Fra HACS 2.0 vises opdateringer også i Home Assistants normale opdateringsoversigt.

## Fejlfinding

### HACS vises ikke efter installation

- Sørg for at du har genstartet Home Assistant
- Ryd browser-cache (Ctrl+F5)
- Tjek at HACS-filerne ligger i `custom_components/hacs/`

### GitHub API limit

HACS bruger GitHub API'en, som har begrænsninger. Hvis du får fejl:

- Vent 1 time og prøv igen
- Sørg for at din GitHub-konto er korrekt forbundet

### Integration virker ikke efter download

1. Tjek at du har genstartet Home Assistant
2. Tjek loggen under **Indstillinger → System → Logs**
3. Læs integrationens dokumentation på GitHub
4. Søg efter løsninger i Home Assistant Community forum
