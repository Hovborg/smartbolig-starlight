## 🧭 Kom hurtigt i gang

<Aside type="note" title="Anbefalet rækkefølge">
  1. **Vælg platform**: Raspberry Pi (nemt), Proxmox (avanceret), eller Docker (eksisterende server).
  2. **Tilføj Zigbee**: Zigbee2MQTT til sensorer og kontakter fra IKEA, Aqara m.fl.
  3. **Byg egne enheder**: ESP32 med ESPHome til temperatur, bevægelse, LED strips.
  4. **Automatiser**: Start med simple automationer og udvid efterhånden.
</Aside>

## ✅ Om SmartBolig.net

<Aside type="tip" title="Dansk kvalitet">
  Alle guides er skrevet på dansk og testet på rigtige opsætninger. Vi fokuserer på **lokal kontrol** - ingen cloud-afhængighed, dine data bliver hjemme.
</Aside>

- 🔍 Brug **søgefeltet** i toppen til hurtigt at finde guides
- 📋 **Kopiér kodeblokke** direkte – de er klar til brug
- 🇩🇰 Alt indhold på dansk med danske priser og forhandlere
- 🔒 Fokus på privatliv og lokal kontrol
