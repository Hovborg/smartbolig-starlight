## 1. Ophavsret & Ejerskab

### Alt indhold er beskyttet

Alt indhold på SmartBolig.net er beskyttet af ophavsretsloven og tilhører SmartBolig.net, medmindre andet er angivet.

Dette inkluderer, men er ikke begrænset til:
- ✍️ Artikler, guides og tutorials
- 📸 Billeder, diagrammer og illustrationer
- 💻 Kodeeksempler og YAML-konfigurationer
- 🎨 Logoer, design og layout
- 📊 Tabeller og sammenligninger

## 2. Hvad du MÅ gøre

### ✅ Tilladt brug

Du må gerne:

| Handling | Tilladt | Betingelse |
|----------|---------|------------|
| Læse og bruge viden | ✅ Ja | - |
| Dele links til sider | ✅ Ja | Link til originalen |
| Citere korte uddrag | ✅ Ja | Max 100 ord + kildeangivelse |
| Bruge til personligt brug | ✅ Ja | Ikke kommercielt |
| Udskrive til eget brug | ✅ Ja | Ikke redistribuering |
| Bruge kodeeksempler | ✅ Ja | I egne projekter |

### Sådan citerer du korrekt

Hvis du vil citere fra SmartBolig.net:

```markdown
"[Dit citat her]"
— Kilde: SmartBolig.net (https://smartbolig.net/da/...)
```

## 3. Hvad du IKKE MÅ gøre

### ❌ Forbudt brug

Følgende er **strengt forbudt** uden skriftlig tilladelse:

| Handling | Status | Konsekvens |
|----------|--------|------------|
| Kopiere hele artikler | ❌ Forbudt | DMCA takedown |
| Republisere indhold | ❌ Forbudt | Juridisk handling |
| Bruge til AI-træning | ❌ Forbudt | Blokering |
| Sælge eller videresælge indhold | ❌ Forbudt | Erstatningskrav |
| Fjerne copyright-notits | ❌ Forbudt | Juridisk handling |
| Scrape eller crawle automatisk uden tilladelse | ❌ Forbudt | IP-blokering |
| Ændre og udgive som eget | ❌ Forbudt | DMCA + erstatning |

### AI-træning og AI-crawlers

<Aside type="danger" title="AI-træning forbudt">
Brug af indhold fra SmartBolig.net til træning af AI-modeller (herunder LLMs, chatbots, generativ AI) er **udtrykkeligt forbudt**.

Vi blokerer rene trænings-, dataset- og scraping-crawlers via robots.txt. AI-søge- og assistant-crawlers, der henter indhold på brugerens anmodning og kan sende trafik tilbage til SmartBolig.net, kan være tilladt.
</Aside>

## 5. Affiliate Links

### Kommercielle links

SmartBolig.net indeholder affiliate links til produkter. Når du køber via disse links:

- Vi modtager en lille kommission
- Prisen for dig er uændret
- Links er markeret med "Affiliate" eller lignende

Se vores fulde [Affiliate Disclosure](/da/juridisk/affiliate-disclosure/) for detaljer.

Vi anbefaler kun produkter vi selv bruger eller ville bruge. Dog:

## 6. Ansvarsfraskrivelse

### Brug på eget ansvar

Indholdet på SmartBolig.net leveres "som det er":

- **Ingen garanti** for nøjagtighed eller fuldstændighed
- **Ingen ansvar** for skader ved brug af guides
- **Elektricitet er farligt** - brug altid autoriseret elektriker
- **Smart home** kan fejle - hav altid backup-løsninger

<Aside type="caution" title="Elektricitet">
Arbejde med 230V installation skal ALTID udføres af autoriseret elektriker. SmartBolig.net er ikke ansvarlig for skader ved uautoriseret arbejde.
</Aside>

## 8. Overtrædelser

### Hvad sker der ved overtrædelse?

Ved overtrædelse af disse vilkår kan vi:

1. Sende cease and desist-brev
2. Indgive DMCA takedown-anmodning
3. Blokere IP-adresser
4. Tage juridiske skridt
5. Kræve erstatning

### Rapporter overtrædelser

Har du fundet uautoriseret kopiering af vores indhold?

Kontakt os på: **kontakt@smartbolig.net**

Inkluder venligst:
- URL til det kopierede indhold
- URL til original på SmartBolig.net
- Bevis for kopiering (screenshot)
