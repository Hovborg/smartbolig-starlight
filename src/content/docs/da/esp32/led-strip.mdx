Adresserbare LED strips giver fantastiske lys-effekter. Hver LED kan styres individuelt med millioner af farver!

## 📋 Forudsætninger

- ✅ [ESP32 sat op med ESPHome](/da/esp32/kom-godt-i-gang/)
- ✅ Loddekolbe (eller strips med stik)
- ✅ 5V strømforsyning (mindst 3A)

## 🛒 Komponenter

| Del | Beskrivelse | Pris ca. |
|-----|-------------|----------|
| ESP32 DevKit | WiFi mikrocontroller | 45 kr |
| WS2812B LED strip | 1m, 60 LEDs/m | 60 kr |
| 5V strømforsyning | Mindst 3A | 50 kr |
| Ledninger | Til strøm + signal | 10 kr |
| **Total** | | **~165 kr** |

<Aside type="caution" title="⚡ Strømforsyning er vigtig!">
LED strips bruger MEGET strøm:
- 60 LEDs fuld hvid = ~3.6A
- 144 LEDs fuld hvid = ~8.6A

**Brug IKKE USB** - det kan max levere 0.5-1A. Køb en dedikeret 5V strømforsyning.
</Aside>

## 🎨 LED Strip typer

| Type | Farver | Anbefalet |
|------|--------|-----------|
| **WS2812B** | RGB (16M farver) | ⭐ God til de fleste |
| **SK6812** | RGBW (med ægte hvid) | ⭐⭐ Bedre hvid |
| **WS2811** | RGB (12V version) | Lange strips |

## 🔌 Tilslutning

### Wiring diagram

![LED Strip til ESP32 wiring diagram](/images/guides/led-strip/wiring.svg)

<Steps>
1. **5V** fra strømforsyning → LED strip **+5V**
   
   <Aside type="danger">
   Forbind IKKE 5V til ESP32! Den får strøm fra USB.
   </Aside>

2. **GND** fra strømforsyning → LED strip **GND** OG ESP32 **GND**
   
   <Aside type="caution">
   Fælles GND er **kritisk**! Uden det virker data-signalet ikke.
   </Aside>

3. **GPIO16** fra ESP32 → LED strip **DIN** (Data In)
</Steps>

<Aside type="tip" title="Farverækkefølge (GRB/RGB)">
Hvis farverne er forkerte (f.eks. rød viser grøn), prøv at ændre `type:` fra `GRB` til `RGB` eller `BRG`.
</Aside>

## 📤 Upload og Test

<Steps>
1. Klik **INSTALL** → **Wirelessly**

2. Gå til **Home Assistant** → **Oversigt**

3. Find din LED strip og tænd den

4. Prøv at ændre farve og vælg effekter
</Steps>

## 🤖 Automation idéer

### Tænd ved solnedgang

```yaml title="automations.yaml"
automation:
  - alias: "Ambient lys ved solnedgang"
    trigger:
      - platform: sun
        event: sunset
    action:
      - service: light.turn_on
        target:
          entity_id: light.led_strip
        data:
          brightness_pct: 50
          rgb_color: [255, 180, 100]  # Varm hvid
```

### Rød ved alarm

```yaml title="automations.yaml"
automation:
  - alias: "Rød alarm"
    trigger:
      - platform: state
        entity_id: alarm_control_panel.home
        to: "triggered"
    action:
      - service: light.turn_on
        target:
          entity_id: light.led_strip
        data:
          effect: "Strobe"
          rgb_color: [255, 0, 0]
```

## 🔧 Fejlfinding

<Aside type="tip" title="LEDs lyser ikke?">
- Tjek at GND er forbundet mellem ESP32 og strømforsyning
- Tjek at strømforsyning er tændt
- Prøv kun de første 10 LEDs (reducer `num_leds`)
- Tjek at DIN (ikke DOUT) er forbundet
</Aside>

<Aside type="tip" title="Forkerte farver?">
- Ændre `type:` mellem `GRB`, `RGB`, `BRG`
- WS2812B er typisk GRB
- SK6812 kan være GRBW eller RGBW
</Aside>

<Aside type="tip" title="Flimren eller random farver?">
- Fælles GND mangler - forbind ESP32 GND til strømforsyning GND
- Dårligt signal - prøv kortere kabel til DIN
- For lang strip - injicer 5V i midten af strippen
</Aside>
