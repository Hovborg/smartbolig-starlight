ESP32 er en lille mikrocontroller der koster under 50 kroner og kan forbindes til dit WiFi. Med ESPHome kan du lave dine egne sensorer, kontakter og displays - helt uden programmering.

<Aside type="note" title="Open Source">
ESPHome er open source! Se [ESPHome på GitHub](https://github.com/esphome/esphome) for dokumentation og eksempler.
</Aside>

## 🛒 Hvad skal du bruge?

<Aside type="tip" title="Startpakke (~100 kr)">
- **ESP32-WROOM-32 DevKit** - Anbefalet til begyndere
- **USB-kabel** (micro-USB eller USB-C)
- **Home Assistant** kørende på din server
</Aside>

### Anbefalede boards

| Board | Pris | Bedst til |
|-------|------|-----------|
| ESP32-WROOM-32 DevKit | 40-50 kr | Begyndere |
| ESP32-C3 Super Mini | 25 kr | Små projekter |
| ESP32-S3 | 60 kr | Displays, kamera |
| D1 Mini (ESP8266) | 25 kr | Simple sensorer |

<Aside type="note" title="Hvor køber jeg?">
- **Hurtigt:** Proshop, av-cables.dk, elektronik-lavpris.dk
- **Billigt:** AliExpress (2-3 ugers levering)
</Aside>
