## 🧠 Sådan virker automationer

Alle automationer har tre dele:

```
TRIGGER  →  BETINGELSE  →  HANDLING
(Hvornår?)   (Hvis?)       (Hvad?)
```

| Del | Beskrivelse | Eksempler |
|-----|-------------|-----------|
| **Trigger** | Hvad starter automationen | Solnedgang, bevægelse, tid, temperatur |
| **Betingelse** | Hvad skal være sandt | Er nogen hjemme? Er det mørkt? |
| **Handling** | Hvad der sker | Tænd lys, send besked, start musik |

## 📚 Automation idéer

### 🏠 Daglig brug

| Automation | Trigger | Handling |
|------------|---------|----------|
| Lys ved solnedgang | Sol går ned | Tænd stuelys |
| Godmorgen rutine | Kl. 07:00 på hverdage | Tænd lys, start kaffe |
| Velkomst hjem | Person ankommer | Tænd lys og varme |
| Godnat | Kl. 23:00 | Sluk alt, lås døre |

### 🔔 Notifikationer

| Automation | Trigger | Handling |
|------------|---------|----------|
| Vaskemaskine færdig | Strømforbrug < 5W | Push besked |
| Dør glemt åben | Dør åben > 10 min | Push besked |
| Lav temperatur | Temp < 18°C | Push besked |

### 💡 Avanceret

| Automation | Trigger | Handling |
|------------|---------|----------|
| Adaptivt lys | Tid på dagen | Juster farvetemperatur |
| Ferietilstand | Ingen hjemme 24h+ | Simuler tilstedeværelse |
| Energi overvågning | Høj effekt | Log og advar |

## 🔧 Nyttige værktøjer

### Hjælpere (Helpers)

Gå til **Indstillinger** → **Enheder og tjenester** → **Hjælpere** for at oprette:

| Hjælper | Brug til |
|---------|----------|
| **Input Boolean** | Ferietilstand, gæstetilstand |
| **Input Number** | Temperaturgrænser, timere |
| **Input Select** | Hustilstand (hjemme/væk/sover) |
| **Timer** | Countdown til handlinger |

### Scener

Kombinér flere handlinger:

```yaml title="scenes.yaml"
- name: "Hygge aften"
  entities:
    light.stue:
      state: on
      brightness: 102
      color_temp: 400
    light.tv_baggrund:
      state: on
      rgb_color: [255, 100, 50]
    media_player.stue:
      state: playing
```
