### 2. Claude Code releases: v2.1.212

**Hvad ændrede sig:** Claude Code releases offentliggjorde opdateringen 17. jul. 2026. Claude Code v2.1.212 ændrer /fork, så den kopierer samtalen ind i en ny baggrundssession med sin egen række under claude agents, mens den gamle in-session-subagent-funktion flyttes til en ny kommando, /subtask. Udgivelsen tilføjer en claude auto-mode reset-kommando med bekræftelsesprompt &#40;kan springes over med --yes&#41;, en sessionsbred grænse for WebSearch-kald på som standard 200 &#40;kan justeres via en miljøvariabel&#41;, samt et loft over subagenter pr. session.

**Hvorfor det er relevant:** For dem der bruger Claude Code som kodeagent hjemme, begrænser dette uventede søgeløkker og antallet af subagenter en session kan starte, hvilket bør give mere forudsigeligt API-forbrug. **Sådan verificerer du det:** Opdater til v2.1.212, afprøv /fork og /subtask én gang hver, og tjek at CLAUDE&#95;CODE&#95;MAX&#95;WEB&#95;SEARCHES&#95;PER&#95;SESSION afspejler den grænse, du forventer.
