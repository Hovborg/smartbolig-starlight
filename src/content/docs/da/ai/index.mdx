Denne del af SmartBolig.net handler om AI-værktøjer som en selvstændig kategori. Den er adskilt fra Home Assistant, ESP32, produkter og smart home-guides, så AI-indholdet ikke bliver blandet sammen med resten af siden.

Første fokus er **AI CLI'er**: terminal-værktøjer som kan læse kode, forklare projekter, skrive ændringer og køre kommandoer med dine tilladelser.

<Aside type="caution" title="Start sikkert">
AI CLI'er kan få adgang til filer, terminalkommandoer og nogle gange netværk. Start altid i et Git-repo, tjek `git status`, og begynd med læse- eller forklaringsopgaver før du giver tilladelse til ændringer.
</Aside>

## Anbefalet rækkefølge

1. Læs [AI CLI'er: kom godt i gang](/da/ai/ai-cli/)
2. Vælg én CLI og lav en første læseopgave.
3. Læs [sikkerhed og permissions](/da/ai/ai-cli/sikkerhed-og-permissions/) før du lader værktøjet ændre filer.
4. Læs [priser og planer](/da/ai/ai-cli/priser-og-planer/), hvis du vil bruge API keys eller arbejde meget i store repos.
5. Opret en projekt-instruktionsfil, når du har fundet et workflow der virker.
6. Følg [AI-nyheder](/da/ai/nyheder/) for ændringer i modeller, CLI'er, API-priser og sikkerhed.
