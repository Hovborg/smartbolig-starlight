## 🎯 Vores mission

SmartBolig.net har ét mål: At gøre smart home tilgængeligt for alle danskere.

Vi tror på at:
- **Privatliv kommer først** - Derfor fokuserer vi på lokale løsninger som Home Assistant
- **Du ikke behøver være ekspert** - Vores guides er skrevet så alle kan følge med
- **Open source er fremtiden** - Gratis software, ingen abonnementer, fuld kontrol

<Aside type="tip" title="Vores filosofi">
Vi anbefaler kun produkter og løsninger vi selv bruger eller ville bruge. Ingen sponsorerede anmeldelser, ingen skjulte agendaer.
</Aside>

## 👨‍💻 Hvem står bag?

SmartBolig.net drives af entusiaster med passion for smart home og open source teknologi.

Vi har selv bygget vores smart homes fra bunden med Home Assistant, og vi deler vores erfaringer - både succeser og fejltagelser.

**Erfaring med:**
- Home Assistant siden 2020
- 100+ Zigbee enheder
- 20+ ESP32 projekter
- Proxmox, Docker, og diverse hosting setups

## 💰 Hvordan tjener vi penge?

SmartBolig.net er gratis at bruge. Vi tjener penge gennem:

1. **Affiliate links** - Når du køber produkter via vores links, får vi en lille kommission (det koster dig ikke ekstra)
2. **Annoncer** - Vi viser begrænsede, relevante annoncer via Google AdSense og Ezoic

Vi lader **aldrig** indtjening påvirke vores anbefalinger. Læs mere i vores [affiliate disclosure](/da/juridisk/affiliate-disclosure/).
