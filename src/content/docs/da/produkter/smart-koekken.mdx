    ```yaml
    # HACS → Integrations → Alexa Media Player
    
    # Eksempel: Morgenautomastion
    automation:
      - alias: "Kaffe - Start ved alarm"
        trigger:
          - platform: state
            entity_id: binary_sensor.alarm_clock
            to: "on"
        condition:
          - condition: time
            weekday:
              - mon
              - tue
              - wed
              - thu
              - fri
        action:
          - service: media_player.play_media
            entity_id: media_player.kitchen_echo
            data:
              media_content_type: custom
              media_content_id: "Alexa, brew coffee"
    ```

- ✅ WiFi + Instant Brands Connect app
- ✅ 800+ guidede opskrifter
- ✅ Fjernmonitorering

### Home Assistant Integration

```yaml
# Via Instant Brands Connect integration
# (Begrænset officiel support - brug via stemmeasistenter)

# Alternativ: Smart stikkontakt til on/off
switch:
  - platform: template
    switches:
      instant_pot:
        friendly_name: "Instant Pot"
        value_template: "{{ states('switch.smart_plug_instant_pot') }}"
        turn_on:
          service: switch.turn_on
          entity_id: switch.smart_plug_instant_pot
        turn_off:
          service: switch.turn_off
          entity_id: switch.smart_plug_instant_pot

# Notifikation når strømforbrug falder (madlavning færdig)
automation:
  - alias: "Instant Pot færdig"
    trigger:
      - platform: numeric_state
        entity_id: sensor.instant_pot_power
        below: 5
        for: "00:05:00"
    condition:
      - condition: numeric_state
        entity_id: sensor.instant_pot_power
        above: 0
    action:
      - service: notify.mobile_app
        data:
          title: "🍲 Madlavning færdig"
          message: "Instant Pot er færdig med at tilberede"
```

    ### MEATER Integration
    
    Via **MEATER HACS integration**:
    
    ```yaml
    # HACS → Integrations → MEATER
    # Kræver MEATER Cloud konto
    
    # Entiteter:
    # - sensor.meater_probe_internal_temp
    # - sensor.meater_probe_ambient_temp
    # - sensor.meater_probe_target_temp
    # - sensor.meater_cook_status

                Temperatur: {{ states('sensor.meater_probe_internal_temp') }}°C
                Tag det af varmen og lad det hvile.
          - action: tts.speak
            target:
              entity_id: tts.google_translate_da_dk
            data:
              media_player_entity_id: media_player.kitchen_speaker
              message: "Kødet er klar. Tag det af varmen nu."
    ```

### Home Assistant

```yaml
# Via Anova integration (community)
# HACS → Integrations → Anova

# Entiteter:
# - climate.anova_cooker
# - sensor.anova_current_temp
# - sensor.anova_target_temp

# Eksempel: Start sous vide på vej hjem
automation:
  - alias: "Start sous vide ved afgang fra arbejde"
    trigger:
      - platform: zone
        entity_id: person.brian
        zone: zone.work
        event: leave
    condition:
      - condition: state
        entity_id: input_boolean.sous_vide_scheduled
        state: "on"
    action:
      - service: climate.set_temperature
        entity_id: climate.anova_cooker
        data:
          temperature: 57  # Perfekt bøf
      - service: notify.mobile_app
        data:
          title: "🥩 Sous vide startet"
          message: "Anova varmer op til 57°C"
```

## 🏠 Smarte Automationer

### Morgenrutine

```yaml
automation:
  - alias: "Køkken morgenrutine"
    trigger:
      - platform: time
        at: input_datetime.morning_alarm
    condition:
      - condition: time
        weekday:
          - mon
          - tue
          - wed
          - thu
          - fri
    action:
      # Tænd køkkenlys
      - service: light.turn_on
        entity_id: light.kitchen
        data:
          brightness_pct: 70
          color_temp: 350
      # Start kaffemaskine (via smart plug)
      - service: switch.turn_on
        entity_id: switch.coffee_maker
      # Afspil nyheder
      - delay: "00:00:30"
      - service: media_player.play_media
        entity_id: media_player.kitchen_speaker
        data:
          media_content_type: news
          media_content_id: "flash briefing"
```

### Forvarm Ovn ved Hjemkomst

```yaml
automation:
  - alias: "Forvarm ovn på vej hjem"
    trigger:
      - platform: zone
        entity_id: person.brian
        zone: zone.home
        event: enter
    condition:
      - condition: state
        entity_id: input_boolean.dinner_planned
        state: "on"
      - condition: time
        after: "16:00:00"
        before: "19:00:00"
    action:
      - service: notify.mobile_app
        data:
          title: "🔥 Forvarm ovn?"
          message: "Du er næsten hjemme. Skal ovnen forvarmes?"
          data:
            actions:
              - action: PREHEAT_OVEN
                title: "Ja, 200°C"
              - action: SKIP_PREHEAT
                title: "Nej tak"
```

### Madlavning Færdig Notifikation

```yaml
automation:
  - alias: "Komfur slukket - mad færdig"
    trigger:
      - platform: numeric_state
        entity_id: sensor.stove_power
        below: 10
        for: "00:02:00"
    condition:
      - condition: template
        value_template: >
          {{ states('sensor.stove_power_today') | float > 0.3 }}
    action:
      - service: notify.mobile_app
        data:
          title: "👨‍🍳 Komfuret er slukket"
          message: "Madlavning ser ud til at være færdig"
```
