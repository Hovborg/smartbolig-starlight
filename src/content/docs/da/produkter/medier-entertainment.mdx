## 🥇 Apple TV 4K

<Tabs>
  <TabItem label="Oversigt">
    ### Hvorfor Apple TV 4K?
    
    **Pris:** ~1.200 kr / $180
    
    Apple TV 4K er den **mest polerede** streaming oplevelse:
    
    - ✅ Bedste UI og performance
    - ✅ Ingen reklamer i interface
    - ✅ AirPlay til iPhone/Mac casting
    - ✅ HomeKit hub funktion
    - ✅ Apple Arcade gaming
    - ✅ Siri stemme kontrol
    - ✅ tvOS apps (Netflix, Disney+, osv.)
    
    **Ulemper:**
    - ❌ Dyr
    - ❌ Kræver Apple økosystem for bedste oplevelse
    - ❌ Ingen Plex server funktion
  </TabItem>
  <TabItem label="Home Assistant">
    ### HA Integration
    
    Apple TV integreres via **Apple TV integration**:
    
    ```yaml
    # Autodiscovery - eller manuel tilføjelse
    # Indstillinger → Enheder → Tilføj Integration → Apple TV
    
    # Entiteter:
    # - media_player.apple_tv_stue
    # - remote.apple_tv_stue
    
    # Eksempel automation
    automation:
      - alias: "Dæmp lys ved film"
        trigger:
          - platform: state
            entity_id: media_player.apple_tv_stue
            to: "playing"
        condition:
          - condition: template
            value_template: "{{ state_attr('media_player.apple_tv_stue', 'media_content_type') == 'movie' }}"
        action:
          - service: light.turn_on
            entity_id: light.stue
            data:
              brightness_pct: 10
    ```
  </TabItem>
</Tabs>

**Køb:** [Apple.dk](https://apple.com/dk), [Elgiganten](https://elgiganten.dk), [Power](https://power.dk)

## 🥈 NVIDIA Shield TV Pro

<Tabs>
  <TabItem label="Oversigt">
    ### Hvorfor Shield TV Pro?
    
    **Pris:** ~1.500 kr / $200
    
    Shield TV Pro er **power users favorit**:
    
    - ✅ Plex Media Server indbygget
    - ✅ AI upscaling (HD → 4K)
    - ✅ Dolby Vision + Atmos
    - ✅ GeForce NOW cloud gaming
    - ✅ Kodi support
    - ✅ 2x USB 3.0 porte
    - ✅ 16GB storage + 3GB RAM
    - ✅ Android TV med alle apps
    
    **Ulemper:**
    - ❌ Dyr
    - ❌ Gammel hardware (2019)
    - ❌ Ukendt hvornår næste generation kommer
  </TabItem>
  <TabItem label="Plex Server">
    ### Shield som Plex Server
    
    Shield TV Pro kan køre **fuld Plex Media Server**:
    
    1. Installer Plex Media Server fra Play Store
    2. Tilslut ekstern USB harddisk
    3. Tilføj mediebiblioteker
    4. Stream til alle enheder i hjemmet
    
    **Fordele vs. NAS:**
    - Lavt strømforbrug (~15W)
    - Altid tændt
    - Hardware transcoding
    - Simpelt setup
    
    **Begrænsninger:**
    - Kun 2 USB porte
    - Begrænset storage
  </TabItem>
  <TabItem label="Home Assistant">
    ### HA Integration
    
    Shield integreres via **Android TV integration**:
    
    ```yaml
    # Autodiscovery via Android TV
    # Indstillinger → Enheder → Tilføj Integration → Android TV
    
    # Entiteter:
    # - media_player.shield_tv
    # - remote.shield_tv
    
    # Start Plex film
    automation:
      - alias: "Film aften"
        trigger:
          - platform: event
            event_type: call_service
            event_data:
              domain: script
              service: film_aften
        action:
          - service: media_player.select_source
            target:
              entity_id: media_player.shield_tv
            data:
              source: "Plex"
    ```
  </TabItem>
</Tabs>

**Køb:** [Amazon.de](https://amazon.de), [Nvidia.com](https://nvidia.com)

## 🆓 Jellyfin - Gratis & Open Source

<Tabs>
  <TabItem label="Oversigt">
    ### Hvorfor Jellyfin?
    
    **Pris:** 100% Gratis
    
    Jellyfin er det **bedste gratis alternativ** til Plex:
    
    - ✅ Helt gratis - ingen premium tier
    - ✅ Open source
    - ✅ Ingen tracking
    - ✅ Self-hosted (fuld kontrol)
    - ✅ Hardware transcoding gratis
    - ✅ Live TV og DVR
    - ✅ Home Assistant integration
    
    **Ulemper:**
    - ❌ Færre polerede client apps
    - ❌ Remote access kræver mere opsætning
    - ❌ Mindre automatisk metadata
  </TabItem>
  <TabItem label="Installation">
    ### Docker Installation
    
    ```yaml
    # docker-compose.yml
    version: "3"
    services:
      jellyfin:
        image: jellyfin/jellyfin
        container_name: jellyfin
        network_mode: host
        volumes:
          - /path/to/config:/config
          - /path/to/cache:/cache
          - /path/to/media:/media
        restart: unless-stopped
        # Hardware acceleration (Intel)
        devices:
          - /dev/dri:/dev/dri
    ```
    
    ### Home Assistant Add-on
    
    Jellyfin findes også som **HACS Add-on** til nem installation.
  </TabItem>
  <TabItem label="Home Assistant">
    ### HA Integration
    
    ```yaml
    # Indstillinger → Enheder → Tilføj Integration → Jellyfin
    
    # Kræver:
    # - Jellyfin server URL
    # - API nøgle (Dashboard → API Keys)
    
    # Entiteter:
    # - media_player.jellyfin_stue_tv
    # - sensor.jellyfin_watching
    
    # Automation eksempel
    automation:
      - alias: "Pause Jellyfin ved dørklokke"
        trigger:
          - platform: state
            entity_id: binary_sensor.doorbell
            to: "on"
        condition:
          - condition: state
            entity_id: media_player.jellyfin_stue_tv
            state: "playing"
        action:
          - service: media_player.media_pause
            entity_id: media_player.jellyfin_stue_tv
    ```
  </TabItem>
</Tabs>

<Tabs>

    - ✅ Plex Discover (streaming guide)

    - Live TV & DVR
    - Offline sync
    - Intro skip
    
    **Ulemper:**
    - ❌ Bedste features kræver betaling
    - ❌ Cloud-afhængig auth
    - ❌ Tracking af visninger
  </TabItem>
  <TabItem label="Home Assistant">
    ### HA Integration
    
    ```yaml
    # Plex opdages automatisk
    # Indstillinger → Enheder → Tilføj Integration → Plex
    
    # Entiteter per klient:
    # - media_player.plex_stue_tv
    # - media_player.plex_sovevaerelse
    
    # Sensor for hvad der afspilles
    # - sensor.plex_watching
    
    # Automation: Film lys
    automation:
      - alias: "Bio-tilstand"
        trigger:
          - platform: state
            entity_id: media_player.plex_stue_tv
            to: "playing"

        condition:
          - condition: template
            value_template: "{{ trigger.to_state.attributes.media_content_type == 'movie' }}"
    ```

## 🏠 Medie Automationer

### Basis: Lys ved afspilning

```yaml
automation:
  - alias: "Dæmp lys ved TV afspilning"
    trigger:
      - platform: state
        entity_id: media_player.stue_tv
        to: "playing"
    action:
      - service: light.turn_on
        entity_id: light.stue
        data:
          brightness_pct: 20
          transition: 5

  - alias: "Gendan lys ved pause/stop"
    trigger:
      - platform: state
        entity_id: media_player.stue_tv
        from: "playing"
    action:
      - service: light.turn_on
        entity_id: light.stue
        data:
          brightness_pct: 80
          transition: 3
```

### Avanceret: Bio Scene

```yaml
# scenes.yaml
scene:
  - name: Bio Tilstand
    entities:
      light.stue:
        state: on
        brightness_pct: 5
        color_temp: 500
      light.tv_baggrund:
        state: on
        brightness_pct: 30
        rgb_color: [50, 50, 100]
      cover.stue_gardiner:
        state: closed

automation:
  - alias: "Aktiver bio ved film start"
    trigger:
      - platform: state
        entity_id: media_player.plex_stue
        to: "playing"
    condition:
      - condition: template
        value_template: >
          {{ state_attr('media_player.plex_stue', 'media_content_type') == 'movie' }}
      - condition: sun
        after: sunset
    action:
      - service: scene.turn_on
        entity_id: scene.bio_tilstand
```

### TTS Announcements

```yaml
automation:
  - alias: "Annoncering ved dørklokke"
    trigger:
      - platform: state
        entity_id: binary_sensor.doorbell
        to: "on"
    action:
      # Pause musik
      - service: media_player.media_pause
        entity_id: media_player.sonos_stue
      # Annoncering
      - action: tts.speak
        target:
          entity_id: tts.google_translate_da_dk
        data:
          media_player_entity_id: media_player.sonos_stue
          message: "Der er nogen ved døren"
      # Vent og genoptag
      - delay: "00:00:05"
      - service: media_player.media_play
        entity_id: media_player.sonos_stue
```
