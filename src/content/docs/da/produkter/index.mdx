## 🔌 Protokol sammenligning

| Protokol | Fordele | Ulemper |
|----------|---------|---------|
| **Zigbee** | Mesh, lavt strømforbrug, mange enheder | Kræver koordinator |
| **WiFi** | Ingen hub, høj hastighed | Belaster router, højere strømforbrug |
| **Z-Wave** | Meget stabilt, lang rækkevidde | Dyrt, færre enheder |
| **Bluetooth** | Billigt, simpelt | Kort rækkevidde, ingen mesh |
| **Thread/Matter** | Fremtiden, åben standard | Nyt, få enheder endnu |

<Aside type="tip" title="Vores anbefaling">
Start med **Zigbee** - det er billigt, stabilt, og har flest kompatible enheder. En Sonoff dongle + nogle Aqara sensorer er en fantastisk start!
</Aside>
